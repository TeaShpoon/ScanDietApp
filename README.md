# ScanDiet

**ScanDiet** — это Android-приложение, которое помогает пользователям следить за своим питанием, сканируя штрих-коды продуктов и проверяя их состав на наличие нежелательных ингредиентов (аллергенов, добавок или соответствие диетическим привычкам).

## Основные функции

*   **Сканирование штрих-кодов**: Быстрое и точное распознавание штрих-кодов с помощью Google ML Kit.
*   **Анализ состава**: Приложение получает информацию о продукте и выделяет ингредиенты, которые не соответствуют вашим настройкам.
*   **Диетические предпочтения**: Настройка списка аллергенов (глютен, лактоза, орехи и т.д.), пищевых добавок (сахар, натрий) и типов питания (вегетарианство, веганство).
*   **История сканирований**: Сохранение последних проверенных продуктов с возможностью фильтрации по безопасности.
*   **Адаптивный интерфейс**: Поддержка различных размеров экранов и темной темы.

## Технологии

*   **Jetpack Compose**: Современный UI-фреймворк.
*   **CameraX & ML Kit**: Для работы с камерой и распознавания штрих-кодов.
*   **Ktor Client**: Для сетевых запросов к API.
*   **Kotlinx Serialization**: Для обработки JSON-данных.
*   **Material 3**: Современный дизайн компонентов.

## Требования

*   Android SDK 24+
*   Android Studio Ladybug (или новее)
*   Подключение к интернету (для получения данных о продуктах)

## Как запустить

1.  Клонируйте репозиторий.
2.  Откройте проект в Android Studio.
3.  Настройте `API_BASE_URL` в конфигурации сборки.
4.  Запустите приложение на физическом устройстве или эмуляторе с поддержкой камеры.
